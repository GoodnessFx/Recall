# RECALL — Neural Memory Layer

RECALL is a high-craft personal memory operating system that unifies your digital fragments—messages, screenshots, PDFs, notes, and bookmarks—into a single, AI-powered archival interface.

## 🧠 Core Philosophy
The creative internet is fragmented. RECALL serves as the permanent memory layer, indexing your digital soul and resurfacing insights when you need them most.

## ✨ Features
- **Neural Timeline**: A fluid, responsive view of your digital history.
- **AI-Powered Search**: Semantic and conversational exploration of your memories.
- **Smart Resurfacing**: Daily "Neural Sparks" that bring back high-resonance fragments.
- **Automated Connectors**: Synchronize with X, Instagram, YouTube, and more.
- **Privacy First**: Local-first architecture with optional Supabase synchronization.
- **Brutalist Aesthetic**: High-end digital archeology design system.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the neural interface:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Building for Production
```bash
npm run build
```

## 🛠 Tech Stack
- **Framework**: React 18 + Vite 6
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Backend**: Supabase (Optional)
- **Icons**: Lucide React
- **Components**: Radix UI Primitives

## 📁 Project Structure
- `src/components`: UI and application components.
- `src/contexts`: Authentication, Theme, and Data providers.
- `src/hooks`: Custom React hooks for memory management and analytics.
- `src/types`: TypeScript definitions.
- `src/styles`: Global styles and Tailwind configuration.
